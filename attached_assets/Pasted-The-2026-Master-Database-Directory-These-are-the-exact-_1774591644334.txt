2. Teaching the Agents: "The Brain Logic"

You will deploy 4 specialized scripts (Agents) on your cPanel. Here is how to instruct each one via your System Prompt:
Agent Alpha: The Scout (Listing & Discovery)

    Instruction: "Go to the 6 Assembly URLs daily. Compare today's list with yesterday's in the politicians table. If a new name appears or a seat becomes 'Vacant', update the database instantly."

    Catch Logic: Focus on Name, Party, and Constituency strings.

Agent Beta: The Miner (Deep Extraction)

    Instruction: "Visit the profile sub-link for each politician found by Agent Alpha. Extract their 'Contact Details', 'Education', and 'Committee Membership'."

    Catch Logic: Use Python BeautifulSoup to find specific CSS classes like .member-info or .portfolio-table.

Agent Gamma: The Auditor (Asset OCR)

    Instruction: "Download Asset PDFs from ECP or Mirror sites. Extract numerical values for 'Land', 'Cash', and 'Jewelry'. Flag if current year assets grew by >50% compared to last year."

    Catch Logic: Use PyPDF2 or Tabula-py to convert PDF tables into CSV data.

Agent Delta: The Samurai (UI & Summary)

    Instruction: "Use the API Key Pool. When a user selects a profile, combine data from Alpha, Beta, and Gamma. Write a summary that highlights 'The Gold, The Ghost, and The Gap' (Wealth, Hidden Links, and Missing Data)."
